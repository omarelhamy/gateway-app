# Gateway Manager Frontend

React frontend for IoT gateway management.

## Features

- Modern UI with Tailwind CSS
- Gateway management interface
- Device management with modal dialogs
- Real-time validation
- Responsive design

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Technologies

- React 18
- Vite
- Tailwind CSS
- Axios (for API calls)
